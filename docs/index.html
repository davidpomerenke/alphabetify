It starts with all symbols in your native alphabet, then slowly introduces more and more foreign characters. You will get used to them and learn the alphabet without any effort. Try it!

You can optionally specify the input language by adding something like lang=de to the beginning of your input. This will add some minor language-specific optimizations. English is assumed by default, and currently the only other language supported for optimization is German.
